# Gestionnaire de fichier CSV en C

## Description

Ce projet est un programme en C qui permet de gérer un fichier CSV contenant des enregistrements de personnes. Chaque enregistrement contient : nom, prénom, genre et téléphone.

## Fonctionnalités

Le programme propose trois principales fonctionnalités :

1. Ajouter un enregistrement dans le fichier `record.csv`
2. Rechercher un enregistrement et le modifier ou le supprimer
3. Quitter le programme

## Fonctionnement

Le programme lit le fichier ligne par ligne, recherche un mot-clé, puis permet à l’utilisateur de modifier ou supprimer l’enregistrement correspondant. Les modifications sont enregistrées dans un fichier temporaire avant de remplacer le fichier original.
